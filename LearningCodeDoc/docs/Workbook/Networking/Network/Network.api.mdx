### Application Layer

The application layer is the only layer that is closest to the end user.  POST requests and GET Requests. Usually **plaintext**. 
- ex: HTTP, DNS, FTP, SSH

### Presentaiton Layer

This layer is responsible for the translation, encryption, and compression of data. Essentially this is the layer in which the information is transformed to have some type of encryption. 
Sending data in networks heavily involve this layer.

- ex. MPEG, AES, SSH, SSL
